# Aplicación de Clima

La **Aplicación de Clima** permite a los usuarios visualizar el clima actual de una ciudad específica, mostrando la condición meteorológica y la temperatura actual. La aplicación se conecta a la API de OpenWeatherMap para obtener datos meteorológicos en tiempo real.

## Características

- Visualiza el clima mediante el ingreso del nombre de la ciudad, proporcionando la condición meteorológica y la temperatura actual.
- Realizada a través del consumo de la API de OpenWeatherMap.
- Interfaz de usuario intuitiva y fácil de usar.
- Mejora de la funcionalidad de visualización del clima.

## Tecnologías Utilizadas

- **Frontend**: React
- **Backend**: Node.js (si aplica)
- **API**: [OpenWeatherMap](https://openweathermap.org/api)
